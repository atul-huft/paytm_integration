
Paytm QR Code Generator on DQR Device

Step 1: POS Configuration
    Open requirements folder > run Paytm Payments.exe as administrator 

Step 2: Install the required package for cryptographic functionalities:
    pip install pycryptodome

Script to generate QR:
    Enter the required amount and orderID to generate QR in createQR.py
    Run: python createQR.py
